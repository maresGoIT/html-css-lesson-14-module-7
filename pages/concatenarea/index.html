Concatenarea (&)

Un alt avantaj, este operatorul &, care adauga numele clasei parinte, la cea din interiorul acoladelor {}.

In SCSS:

.li {
  background-color: red;
  :hover {color: green;}
  &:hover { color: blue }
}

In CSS:

.li {
  background-color: red;
}
.li :hover {
  color: green;
}
.li:hover {
  color: blue;
}

Imbricarea stilurilor (nesting)

Un alt avantaj, este abilitatea de a adauga stiluri pentru clasa in interiorul altei clase CSS.

In SCSS:

.enlarge {
    font-size: 14px;
    transition: {
        property: font-size;
        duration: 4s;
        delay: 2s;
    }

    &:hover { font-size: 36px; }
}

In CSS:

.enlarge {
    font-size: 14px;
    transition-property: font-size;
    transition-duration: 4s;
    transition-delay: 2s;
}
.enlarge:hover {
    font-size: 36px;
}

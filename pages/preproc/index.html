Preprocesoare CSS

O sintaxa de scriere a stilurilor, care mai apoi, cu ajutorul unui "compilator" transforma codul scris in CSS.

Oficial descris ca "CSS cu superputeri" SCSS ofera o alternativa de scriere a stilurilor cu cateva imbunatatiri fata de standardul CSS. Diferenta majora, este ca acest cod nu poate fi interpretat de browserere, el va trebui transforamt in CSS inainte.

In VSCode, pot sa instalez extensia Live Sass Compiler" , care apoi imi genereaza automat fisierul css pe care il adaug in pagina mea HTML.

Variabile SCSS

O sintaxa de scriere a stilurilor, care mai apoi, cu ajutorul unui "compilator" transforma codul scris in CSS.

Variabilele CSS sunt diferite fata de cele din SCSS.

  • Variabilele in SCSS sunt toate compilate, nu le putem modifica in browser. Variabilele CSS sunt incluse în rezultatul CSS.
  • Variabilele CSS poate avea valori diferite pentru diferite elemente, dar variabile SCSS au o singură valoare la un moment dat.
  • Variabilele SCSS sunt imperative, adică dacă utilizați o variabilă și apoi modificați valoarea acesteia, valoarea folosita în primul caz rămâne la fel. Variabilele CSS sunt declarative, ceea ce înseamnă că dacă dvs modificați valoarea, aceasta va afecta atât utilizările anterioare, cât și utilizările ulterioare.

In SCSS:

$variable: value 1;
.rule-1 {
    value: $variable; // value 1
}

$variable: value 2;
.rule-2 {
    value: $variable; // value 2
}

In CSS:

                --variable: value 1;
                .rule-1 {
                  value: value 2;
                }

                --variable: value 2;
                .rule-2 {
                  value: value 2;
                }
